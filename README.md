# Петрозаводск — исторический атлас

Интерактивная карта-сервис о ключевых исторических местах, событиях и личностях
Петрозаводска: от основания Петровского завода в 1703 году до наших дней.

## Запуск

Зависимостей и сборки нет — это статический сайт (Leaflet и шрифты подключаются с CDN).

```bash
cd /Users/di/Projects/ptz
python3 -m http.server 8000
```

Открыть http://localhost:8000 (можно и просто открыть `index.html` в браузере).

## Что внутри

- **Карта** — 43 точки в восьми категориях (исторические места, памятники, музеи,
  храмы, набережная, утраченное наследие, военная история, окрестности)
  с фильтрами и поиском.
- **Хроника** — 32 события от петроглифов IV тысячелетия до н. э. до 2021 года,
  сгруппированные по эпохам, с фотографиями; события открываются на карте.
- **Личности** — 12 фигур от Петра I и Державина до Гаскойна, Андропова и
  Рождественского, с портретами и перекрёстными ссылками на точки карты.
- **Районы** — 13 исторических районов (пунктирные зоны на карте); история
  каждого встроена в карточку района по периодам (по материалам проекта
  «Литературные адреса Петрозаводска» Петрозаводской ЦБС, без внешних ссылок).
- **О городе** — краткая справка и факты.

Фотографии объектов и портреты подтягиваются из русской Википедии её
пакетным API (по 50 названий за запрос); если статьи или снимка нет,
карточка показывается без фото. Привязки названий статей проверены.

## Структура

```
index.html      — разметка и вкладки
css/styles.css  — стили
js/data.js      — база данных: PLACES, EVENTS, PEOPLE, DISTRICTS, WIKI, CATEGORIES
js/app.js       — карта (Leaflet), фильтры, поиск, вкладки, загрузка фото
```

Чтобы добавить точку, событие, личность или район — достаточно дописать объект
в соответствующий массив в `js/data.js` (фото — добавить запись `id → статья
Википедии` в `WIKI`).

> Координаты точек и центров районов взяты из OpenStreetMap (Overpass API);
> для утраченных зданий и исторической площадки Александровского завода —
> обоснованные приближения. Границы районов условные (круги вокруг
> place-узлов OSM). Даты и факты сверены по общедоступным источникам,
> но для публикации стоит перепроверить.
